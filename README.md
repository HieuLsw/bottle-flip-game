# Bottle Flip Game.
#### Un juego hecho en Javascript ES6 + Webpack + Phaser.

## Instrucciones
- Arrastrar la botella para que gire en el aire y caiga.


# Chequeos
- Que la botella gire antes de caer.
- Que la botella caiga parada o invertida.
